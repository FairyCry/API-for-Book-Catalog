# Book Catalog V3

API для книжного каталога.

Простое API для работы с книгами, с регистрацией пользователей, редактированием книг, а также системой категорий и модерации.

> Проект учебный и сильно условный. Суть проекта — выучить новые технологии.

## Живое API

enchanting-youthfulness-production-e0b0.up.railway.app/api

## Технологии

- TypeScript
- NestJS
- TypeORM
- PostgreSQL
- JWT (авторизация)
- bcrypt (хэширование паролей)
- class-validator (валидация DTO)

## Функционал

### Пользователи
- Регистрация и авторизация (JWT)
- Две роли: `user` и `admin`
- Редактирование своего профиля
- Администратор может редактировать любого пользователя

### Книги
- Добавление книги (только авторизованные пользователи)
- Редактирование и удаление своих книг
- Пагинация и фильтрация по жанру
- Система модерации: новые книги попадают в статус `pending`, администратор одобряет (`approved`) или отклоняет (`rejected`)
- Публично доступны только одобренные книги

### Категории
- Добавление, просмотр и удаление категорий (только администратор)
- Книга может иметь несколько категорий (ManyToMany)

## Запуск

1. Установить зависимости:
```bash
npm install
```

2. Создать `.env` файл:
```env
DB_HOST=localhost
DB_PORT=5432
DB_USERNAME=postgres
DB_PASSWORD=your_password
DB_NAME=book_catalog
JWT_SECRET=your_secret
```

3. Запустить:
```bash
npm run start:dev
```
